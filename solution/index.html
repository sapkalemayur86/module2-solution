My Favourite Languages

Java

  Java is a high-level programming language developed by Sun Microsystems. It was originally designed for developing programs for set-top boxes and handheld devices, but later became a popular choice for creating web applications.

Python

  Python is an interpreted, high-level, general-purpose programming language. Created by Guido van Rossum and first released in 1991, Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects

Js

  JavaScript is a programming language commonly used in web development. While JavaScript is influenced by Java, the syntax is more similar to C and is based on ECMAScript, a scripting language developed by Sun Microsystems.
